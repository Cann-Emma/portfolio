# PROJECTS

### Data Analysis Of Palmer Archipelago Penguins
[Penguins](https://github.com/Cann-Emma/R/blob/main/Penguin_Analysis.ipynb)

An independent analysis of Antarctic penguins, summarizing data on species, sex, culmen length and depth, flipper length, body mass, and island in Palmer Archipelago. This project included data cleaning, exploratory analysis, visualization, regression(simple and logistic),and classification(K Nearest Neighbor).
